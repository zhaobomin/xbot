# Skill Creator

Use this skill when creating or packaging an xbot skill. A skill is a directory
with a required `SKILL.md` file and optional `scripts/`, `references/`, and
`assets/` resource directories.

Keep generated skill folders small: do not add root-level README files or other
auxiliary documents unless they are part of the skill runtime contract.
